# Kanban Board — Frontend

React frontend for the Kanban board application. Built with Vite, Tailwind CSS, and @dnd-kit for drag-and-drop.

Full project overview, features, and setup instructions (including Docker) are in the backend repo: [kanban-project](https://github.com/hamzameto/kanban-project)

## Tech Stack

- React (Vite)
- React Router
- Axios
- Tailwind CSS
- @dnd-kit (drag-and-drop)

## Running locally

Requires Node.js 20+ and the backend running (see [kanban-project](https://github.com/hamzameto/kanban-project) for backend setup).

```bash
npm install
npm run dev
```

The app expects the backend to be running at `http://localhost:8080`.
